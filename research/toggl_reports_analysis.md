# Toggl Reports Analysis

## Report Types Available
1. **Summary** - Overview with charts and breakdowns
2. **Detailed** - Line-by-line time entries
3. **Workload** - Team capacity view
4. **Profitability** - Revenue analysis
5. **My reports** - Saved custom reports

## Summary Report Features
- Date range selector (This week, custom, etc.)
- Filters: Member, Client, Project, Task, Tag, Description, Add filter
- Rounding options (off/on)
- Create invoice button
- Export button
- Save report button
