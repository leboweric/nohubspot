# Toggl vs NHS Visual Comparison - Key Differences

## Toggl Layout (from screenshot):

### Left Sidebar
- Toggl has a LEFT SIDEBAR with sections: TRACK (Overview, Timer), ANALYZE (Reports, Approvals), MANAGE (Projects, Clients, Members, Billable rates, Invoices, Tags, Goals, Integrations)
- NHS has a TOP NAV BAR

### Timer Bar
- Full width, white background
- "What are you working on?" placeholder text
- Project selector, tag selector, billable ($) toggle, duration display, green play button, manual mode toggle
- ALL ON ONE ROW

### Stats Bar
- Below timer: left/right arrows for date navigation, "All dates" dropdown, "TODAY TOTAL 4:04:20", "WEEK TOTAL 6:57:45"
- View toggles: Calendar | List view | Timesheet
- Settings gear icon

### Project Filter Bar
- Below stats: colored bar showing "SCC INTERNAL WORK (TEAM MEETINGS, PROPOSAL/CONTRACT WORK, ETC.)" and "STRATEGIC CONSULTING & COACHING" and "BOAR..."
- These are filter pills in ALL CAPS

### Day Groups
- "Today" with checkbox, total hours on right (4:04:20)
- Entries EXPANDED by default for recent days
- Each entry row: description (left), group count badge (e.g., "2"), colored dot + project name + client name, $, time range, duration
- Entries are GROUPED by description+project — shows a count badge like "2" or "3"

### Entry Row Layout (left to right):
1. Description text (or "Add des..." placeholder)
2. Group count badge (number in a circle, e.g., "2", "3")
3. Colored dot + Project name (in project color) + Client name (in gray)
4. $ billable indicator
5. Time range (e.g., "1:44 PM - 5:48 PM")
6. Duration (e.g., "4:04:20")
7. Play/action buttons on hover

### Key Visual Details:
- Project names are in the PROJECT COLOR (e.g., pink/red for "SCC Internal Work")
- Client name appears AFTER project name in lighter gray
- Grouped entries show a count badge
- Day headers: "Today", "Yesterday", "Wed, 15 Apr", "17 Apr", "Tue, 7 Apr"
- Entries are dense — about 30px row height
- Background is WHITE
- Left sidebar is DARK (dark purple/navy)

## NHS Current Issues vs Toggl:
1. NHS has TOP NAV, not left sidebar — but this is a broader NHS issue, not time-tracking specific
2. NHS day groups are COLLAPSED by default — Toggl shows them EXPANDED for recent days
3. NHS doesn't GROUP entries by description+project — Toggl shows "2" or "3" badge
4. NHS project names are truncated — Toggl shows them in color with client name
5. NHS doesn't have the project filter bar below stats
6. NHS doesn't have Calendar/List view/Timesheet toggle
7. NHS doesn't have date navigation arrows
